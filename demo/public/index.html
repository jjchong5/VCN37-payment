📚 Pay-Per-Book Library

An x402 micropayment paywall — agents pay 1¢ in USDC to read a book, no account needed.

How x402 works, step by step

  1. The buyer requests GET /books/:id with no payment attached.
  2. The server replies HTTP 402 Payment Required with a JSON body describing the price, network, and address to pay.
  3. The buyer's wallet signs an EIP-3009 USDC transfer authorization for that exact amount — no gas needed from the buyer, this is an off-chain signature.
  4. The buyer retries the same request with an X-PAYMENT header carrying that signature.
  5. The server verifies the signature, settles it on-chain (Base Sepolia here), and finally returns the book — with proof of the on-chain transaction in the response.

This whole loop took under a second below. No login, no credit card, no subscription — just a wallet and a price.
