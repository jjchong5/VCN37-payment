📚 Pay-Per-Book Library

An x402 micropayment paywall — connect your own wallet and pay 1¢ in USDC to read a book.

How x402 works, step by step

  1. Your browser requests GET /books/:id with no payment attached.
  2. The server replies HTTP 402 Payment Required with a JSON body describing the price, network, and address to pay.
  3. Your wallet signs an EIP-3009 USDC transfer authorization for that exact amount — no gas needed from you, this is an off-chain signature.
  4. Your browser retries the same request with an X-PAYMENT header carrying that signature.
  5. The server verifies the signature, settles it on-chain (Base Sepolia, a free testnet), and returns the book — with proof of the on-chain transaction.

What you need: a browser wallet (e.g. MetaMask), set to the Base Sepolia testnet, holding a small amount of testnet USDC (free from a faucet — see below). Your private key never leaves your wallet; this page never asks for it.

No testnet USDC yet? Get some free from the Circle USDC faucet (select Base Sepolia).
